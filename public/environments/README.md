# 3D Environment Models

This folder contains 3D environment models for the AI Avatar project.

## Required Models

Download the following models from Sketchfab and place them in this folder:

### 1. School Classroom
- **File**: `classroom-lowpoly.glb`
- **Source**: https://sketchfab.com/3d-models/low-poly-classroom-8b1bf103c90349d981c9c2ae1c0596e5
- **Instructions**:
  1. Visit the link above
  2. Click "Download 3D Model"
  3. Select "glTF Binary (.glb)" format
  4. Save as `classroom-lowpoly.glb` in this folder

### 2. Modern Office
- **File**: `office.glb`
- **Source**: https://sketchfab.com/3d-models/modern-office-interior-a95a81f32bb1461da296d1e2351e7093
- **Instructions**:
  1. Visit the link above
  2. Click "Download 3D Model"
  3. Select "glTF Binary (.glb)" format
  4. Save as `office.glb` in this folder

### 3. Realistic Forest
- **File**: `forest.glb`
- **Source**: https://sketchfab.com/3d-models/realistic-forest-26d5693b975a481284dec4e2a61d06da
- **Instructions**:
  1. Visit the link above
  2. Click "Download 3D Model"
  3. Select "glTF Binary (.glb)" format
  4. Save as `forest.glb` in this folder

## Alternative: School Class Room
If you prefer the alternative classroom model:
- **Source**: https://sketchfab.com/3d-models/school-class-room-371cd1996ed4452ebd1990132ba2d669
- Save as `classroom-alternative.glb`

## File Structure

After downloading, your folder should look like:
```
/public/environments/
├── README.md (this file)
├── classroom-lowpoly.glb
├── office.glb
└── forest.glb
```

## Notes

- All models are from Sketchfab and may require creating a free account to download
- Make sure to download in **glTF Binary (.glb)** format
- The built-in classroom environment will be used if these files are not present
- File sizes may vary (typically 5-50 MB each)
